# yunzai_jsrun

#### 介绍

自己做的Yunzai机器人插件（仅支持v3），可实现在q群内安装插件/查看文件等（可能有点鸡肋？）

#### 安装教程

将js文件复制到/plugins/example文件夹

#### 使用说明

`##+指令`：执行node命令；

`#cmd+指令`：执行cmd指令；

`#js设置(权限|编码|合并转发|超时)`：编辑插件设置（重启恢复默认）
